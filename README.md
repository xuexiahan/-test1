# 乡城文旅软件 - 测试项目

 一个完整的 Web 功能测试 + 接口测试项目，用于展示测试实习生的综合能力。

## 项目简介

乡城文旅是一个旅游规划平台的**前端演示版本**，覆盖美食、交通、住宿、门票四大业务模块。本项目包含：

- 完整的静态网页代码
- 200+ 条功能测试用例
- 100+ 条接口测试用例
- 测试点脑图
- 缺陷报告

##  如何运行项目

### 方法一：直接打开
双击 `index.html` 文件，在浏览器中打开即可。

### 方法二：用 IDE 打开
1. 用 VS Code / IntelliJ IDEA / PyCharm 打开项目文件夹
2. 右键 `index.html` → 选择浏览器打开

### 方法三：本地服务器运行
```bash
# Python 3
python -m http.server 8080

# 访问 http://localhost:8080
